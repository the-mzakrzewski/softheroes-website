# Target
- Website for software development company (one person company)
- Make professional, modern, and user-friendly website
- Mobile first approach

# Design
- Responsive design with clean and minimalistic layout
- Use of Tailwind CSS for utility-first styling
- Avoid using too many colors
- Don't design like another AI made website

# Workflow
- All css classes shold be in app.css file instaed of class attribute
- Don't install any npm packages
- Don't run npm run build command
- Website will be in Polish language

# Tools
- Laravel
- Tailwind CSS
